Function practice:

  1. Write a function that adds two numbers together, the numbers can be variables or parameters.
  2. Write a function that multiplies a number by itself, the result must be returned by the function
  3. Write a function that takes in a parameter of a name and then adds 'hello, ' before the name. Example: 'hello, aaron'. The result must be returned!
  4. Write a function that takes in a string as a parameter, and converts it to uppercase and returns the result. It must be an arrow function!
